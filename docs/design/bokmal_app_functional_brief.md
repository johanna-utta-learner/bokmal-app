# Bokmål App — Functional Brief

## Purpose

Build a simple web app for practising Norwegian Bokmål.

German is used as the interface and prompt language. Bokmål is the language the learner practises and recalls.

The app should focus on A1–A2 Bokmål practice.

## Core Learning Flow

The basic learning flow is:

```text
German prompt → learner recalls or enters Bokmål → app shows correct Bokmål
```

The app should be focused on practice, recall, and repetition rather than explanation.

## Main Functions

The app should allow the learner to:

- practise Bokmål vocabulary
- practise Bokmål nouns in singular and plural forms
- practise Bokmål verbs in infinitive, present, and past tense
- practise German-to-Bokmål sentence translation
- review corrected Bokmål sentences
- add new Bokmål items from lessons, reading, writing, and other study
- revise both built-in content and user-added content

## Main App Sections

The app should include these basic sections:

```text
Start
Wortschatz
Karten
Übungen
Meine Wörter
Fortschritt
```

## Start

The start page should give a simple overview of the learner’s current practice options.

It should provide quick access to:

- flashcards
- vocabulary practice
- sentence practice
- user-added content

## Wortschatz

The vocabulary section should display stored vocabulary items.

It should support nouns and verbs.

Nouns should show:

- German meaning
- Bokmål singular
- Bokmål definite singular
- Bokmål plural
- Bokmål definite plural

Verbs should show:

- German meaning
- Bokmål infinitive
- Bokmål present tense
- Bokmål past tense

## Karten

The flashcard section should let the learner practise recall.

Possible card directions:

- German → Bokmål
- Bokmål → German
- German → specific Bokmål noun form
- German → specific Bokmål verb form

The learner should be able to reveal the answer and move to the next card.

## Übungen

The exercises section should provide simple tests.

Exercise types may include:

- type the Bokmål answer from a German prompt
- choose the correct Bokmål noun form
- choose the correct Bokmål verb form
- complete a sentence
- correct a previously wrong sentence

The app should show the correct answer after the learner responds.

## Meine Wörter

The user-added content section should let the learner add new study material.

The learner should be able to add:

- nouns
- verbs
- sentences
- corrections

User-added material should become available for flashcards and exercises.

## Fortschritt

The progress section should give a simple overview of practice.

It may show:

- number of items practised
- number of cards reviewed
- number of correct answers
- items needing more practice

Progress tracking should stay simple and should not dominate the app.

## Content Rules

The app should not include:

- grammar explanations
- German/Bokmål grammar comparison sections
- long notes
- cluttered teaching panels

The app should stay focused on:

```text
see prompt
recall Bokmål
check answer
repeat
```
